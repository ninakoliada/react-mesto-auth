# Проект: Место

### Технологии

* HTML
* CSS
* JS
* БЭМ
* React

### Структура проекта:

* `src/blocks` - блоки содержащие стили
* `src/images` - картинки и иконки
* `src/vendor` - дополнительные стили и шрифты
* `src/components` - компоненты
* `src/utils` - утилиты

### Запуск

Устанавливаем зависимости - `npm ci`
Запуск проекта - `npm start`
